## HOW THIS FILE CONNECTS TO THE THREE-NODE LOOP

This file is the fourth node — the reflection layer.

BD Project observes: commercial decisions, customer strategy, requirements quality
Claude Code observes: implementation discipline, wrap compliance, feature quality
CONTEXT_LEARNING.md records: what's improving, what's recurring, what needs work

The loop runs: signal → requirements → build → product → customer → signal
The learning runs: session → observation → suggestion → next session → improvement

Both loops compound. The product gets better with each customer signal.
The operator gets better with each session observation.
